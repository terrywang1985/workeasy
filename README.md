# 不正经的员工 - 微信小游戏

一款脑洞解谜 + 职场搞笑风格的微信小游戏

## 项目结构

```
weixin_workeasy/
├── game.js                    # 游戏主入口
├── game.json                  # 游戏配置文件
├── project.config.json        # 项目配置文件
├── js/
│   ├── SceneManager.js       # 场景管理器
│   ├── BaseScene.js          # 基础场景类
│   ├── scenes/               # 场景目录
│   │   ├── MainScene.js      # 主界面场景
│   │   ├── LevelsScene.js    # 选关界面场景
│   │   └── GameScene.js      # 游戏关卡场景
│   └── levels/               # 关卡目录（每关一个文件）
│       ├── BaseLevel.js      # 关卡基类
│       ├── index.js          # 关卡索引（注册表）
│       ├── Level31.js        # 第31关
│       ├── Level32.js        # 第32关（待添加）
│       └── LevelTemplate.js  # 关卡模板
├── README.md                 # 项目说明文档
├── HOW_TO_ADD_LEVEL.md       # 添加关卡教程
└── QUICKSTART.md             # 快速开始指南
```

## 功能特性

### 1. 主界面
- 显示游戏标题 "不正经的员工"
- 简笔画风格的员工形象
- "开始游戏" 按钮
- 版本信息显示

### 2. 选关界面
- 关卡按 31-60 编号排列
- 5 列网格布局
- 第一关（31关）可点击进入
- 其他关卡灰色锁定，显示"敬请期待"
- 点击锁定关卡弹出提示
- 顶部返回按钮和帮助按钮（占位）

### 3. 第一关（31关）- 上班迟到
**主题**：玩家早上迟到，需偷偷进入公司，避开正在门口检查的老板

**场景元素**：
- 公司大门
- 老板（站在门旁，简笔画风格）
- 玩家（门口外侧）
- 扫帚（可点击道具）
- 快递箱（干扰道具）

**解谜逻辑**：
1. 点击扫帚 → 玩家拿起扫帚伪装成清洁工
2. 点击大门 → 成功进入（如已伪装）
3. 未伪装直接点门 → 失败（老板发怒）
4. 点击老板 → 显示提示文字
5. 点击快递箱 → 显示"现在用不上..."

**通关条件**：先点击扫帚，再点击大门

## 技术实现

### 核心技术
- **微信小游戏框架**：使用 Canvas 进行渲染
- **场景管理系统**：支持场景切换、更新和渲染
- **事件系统**：触摸事件处理
- **简笔画绘制**：纯 Canvas API 绘制角色和场景

### 场景管理
- `SceneManager`：统一管理所有场景的切换和生命周期
- `BaseScene`：提供基础的渲染、事件处理和工具方法
- 场景间数据传递通过 `switchScene(name, data)` 实现

### 关卡系统
- **每关一个独立文件**：所有关卡放在 `js/levels/` 目录
- 关卡继承 `BaseLevel` 基类
- 关卡数据和逻辑完全封装在各自文件中
- 通过 `js/levels/index.js` 统一注册和管理
- 支持无限扩展新关卡

## 如何添加新关卡

### 快速添加（3步）

**第1步**：复制模板文件
```bash
复制 js/levels/LevelTemplate.js
重命名为 js/levels/Level32.js
```

**第2步**：编辑关卡内容
```javascript
class Level32 extends BaseLevel {
  constructor() {
    super();
    this.id = 32;
    this.name = '你的关卡名称';
    this.story = '关卡剧情...';
    this.elements = [ /* 场景元素 */ ];
  }
  
  onElementClick(element) {
    // 实现解谜逻辑
  }
}
```

**第3步**：在 `js/levels/index.js` 中注册
```javascript
import Level32 from './Level32.js';  // 导入

const levelRegistry = {
  31: Level31,
  32: Level32,  // 注册
};
```

**详细教程**：请查看 [HOW_TO_ADD_LEVEL.md](HOW_TO_ADD_LEVEL.md)

### 视觉风格
- 配色方案：
  - 背景：`#F5E6D3`（米黄色）
  - 主色：`#8B4513`（棕色）

  - 锁定：`#CCC`（灰色）
- 简笔画角色：圆形头部 + 火柴人身体
- 按钮：白色背景 + 黑色边框
