## 💡 添加新关卡

只需3步：

1. **复制模板**
```bash
cp js/levels/LevelTemplate.js js/levels/Level11.js
```

2. **编辑关卡**
```javascript
this.id = 11;
this.name = '周一会议';
this.story = '周一开会，困死了...';
```

3. **注册关卡**
```javascript
// js/levels/index.js
const Level11 = require('./Level11.js');
const levelRegistry = {
  1: Level01,
  // ...
  11: Level11,  // 添加这行
};
```
